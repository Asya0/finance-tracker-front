# Финансовый трекер

## Описание проекта

Веб-приложение для учета личных финансов, позволяющее пользователю вести учет доходов и расходов, просматривать историю операций и отслеживать текущее финансовое состояние.

## Минимальный функционал

### Авторизация

* Регистрация пользователя.
* Вход в систему.
* Выход из системы.

### Управление операциями

* Добавление доходов.
* Добавление расходов.
* Редактирование операций.
* Удаление операций.
* Просмотр списка всех операций.

### Категории

* Использование категорий для доходов и расходов.
* Привязка категории к каждой операции.

### Главная страница (Дашборд)

* Отображение текущего баланса.
* Общая сумма доходов.
* Общая сумма расходов.
* Последние добавленные операции.

### Фильтрация

* Поиск операций по периоду.
* Фильтрация по категории.
* Фильтрация по типу операции (доход/расход).

### Профиль пользователя

* Просмотр информации о пользователе.
* Изменение пароля.
